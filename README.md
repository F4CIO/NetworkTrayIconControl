# NetworkTrayIconControl
Quickly switch and control your network adapters from Windows tray icon with this WPF application.

![NetworkTrayIconControl](https://github.com/F4CIO/NetworkTrayIconControl/blob/main/CraftSynth.NetworkTrayIconControl.png)

Official page on my blog: http://www.f4cio.com/networktrayiconcontrol

## Features ##
- Enables/disables network adapters on a single click
- On mouse hover over icon shows current Wi-Fi, IP, Mask, DNS, Speed and internet connectivity
- Ability to restart current network adapter or disable all
- Free

## Minimum Requirements ##
- Windows OS (tested on Windows 11 but should work on Windows 7 and 10)
- .Net Framework 8 (if you don't have it already installed go to here
- 3Mb of disk space
